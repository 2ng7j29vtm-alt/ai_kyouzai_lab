二次関数シミュレーター

y = ax² + bx + c のグラフを，スライダーで動かして形の変化を観察しよう。

1. 係数 a, b, c を動かそう
y = 1x² + 0x + 0
・a：グラフの開き方（上向き / 下向き，細い / 広い）
・b：グラフの左右へのずれ方
・c：y切片（x = 0 のときの y の値）
a（開き具合） a = 1.0
b（左右のずれ） b = 0.0
c（y切片） c = 0.0
  • a を動かすと，グラフの「開き方」がどう変わるかに注目しよう。
  • b を動かすと，頂点が左右にどのように動くか見てみよう。
  • c を動かすと，グラフ全体が上下に平行移動することを確認しよう。
2. グラフの変化を観察しよう

※ スマホではスライダーを指で動かして，グラフの形の変化をリアルタイムで確認できます。
